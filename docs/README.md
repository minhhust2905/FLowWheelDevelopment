# FlowWheel

FlowWheel là một tiện ích mạnh mẽ giúp mở rộng chức năng chuột trên Windows, tối ưu cho chuột 5 nút nhưng vẫn hỗ trợ tốt cho mọi loại chuột. Dự án này tập trung vào trải nghiệm cuộn, chuyển đổi tab, điều khiển media, quản lý cửa sổ và nhiều thao tác tiện ích khác chỉ với các tổ hợp phím chuột.

## Tính năng chính
- **Chuyển tab trình duyệt:** RButton + Wheel
- **Đóng tab nhanh:** RButton + MButton
- **Khôi phục tab:** LButton + MButton
- **Cuộn ngang:** RButton + Wheel L/R hoặc Shift + Wheel Up/Down
- **Điều chỉnh âm lượng:** RButton + X1 + Wheel
- **Điều chỉnh độ sáng:** RButton + X2 + Wheel
- **Phóng to/Thu nhỏ:** MButton + Wheel
- **Chuyển/đóng cửa sổ:** X1 + Wheel hoặc RButton + X1 + Wheel
- **Chọn ứng dụng taskbar:** X2 + Wheel
- **Chụp màn hình nhanh:** X1 + X2
- **Ghim cửa sổ (Snap):** RButton + X1/X2
- **Tùy chỉnh sâu qua file cấu hình và giao diện Settings

## Cách sử dụng
1. **Chạy FlowWheel_Main.ahk** (yêu cầu AutoHotkey v2)
2. **Tùy chỉnh**: Nhấn Ctrl+Alt+S để mở Settings, bật/tắt các gesture theo ý muốn.
3. **Xem hướng dẫn nhanh**: Ctrl+Alt+H

## Yêu cầu hệ thống
- Windows 10/11
- [AutoHotkey v2](https://www.autohotkey.com/)

## Cấu trúc dự án
- `src/` - Mã nguồn chính
- `resources/` - Icon, ảnh, âm thanh, config mẫu
- `docs/` - Tài liệu
- `build/` - Script build/pack
